# Deepfake Detection using CNN. 

## To run the app on browser

```
1. Open cmd in the project directory 

2. Download all the requirements via requirements.txt file using command: pip install -r requirements.txt

3. run python -m venv venv : command to create project environment

4. To activate environemnt, run command: venv\Scripts\activate

5. Finally run python app.py to host the project on localhost:5000

6. To end the python environment, run command: deactivate
```
